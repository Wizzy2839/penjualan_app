Tentang WizzyApp

Selamat datang di WizzyApp, solusi toko online sederhana Anda. Proyek ini mendemonstrasikan cara membuat API lokal menggunakan PHP dan MySQL, serta mengintegrasikannya dengan frontend JavaScript modern.

Kami memanfaatkan Tailwind CSS untuk styling dan JavaScript murni untuk menangani semua logika, mulai dari pengambilan data produk, manajemen keranjang belanja, hingga proses checkout.

Backend kami menggunakan PHP untuk REST API yang menangani operasi CRUD ke database MySQL.

Tujuan kami adalah menyediakan aplikasi fungsional dengan tampilan Glassmorphism yang keren.
